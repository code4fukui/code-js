# code-js

highlight code JavaScript, HTML, C

## usage

```html
<script type="module" src="https://code4fukui.github.io/code-js/code-js.js"></script>

<code-js>
onload = () => {
  for (let i = 0; i < 10; i++) {
    console.log(i);
  }
};
</code-js>
```
