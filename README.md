# code-js

- code-js tag to highlight code JavaScript
- also code-html for HTML, code-c for C language

## usage

```html
<script type="module" src="https://js.sabae.cc/code-js.js"></script>

<code-js>
onload = () => {
  for (let i = 0; i < 10; i++) {
    console.log(i);
  }
};
</code-js>
```
